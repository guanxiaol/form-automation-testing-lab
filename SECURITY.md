# Security Policy

## Sensitive data policy

This public version is intentionally sanitized. Do not commit:

- Real accounts, emails, passwords, OTP codes, tokens, cookies or API keys
- `accounts.txt`, `accounts.json`, `register_results.txt`, `emails.txt`
- Runtime logs, browser profiles, local virtual environments or cache files
- Personal machine paths or local absolute paths
- LaunchAgent or auto-start configuration files

## Responsible use

This project is for authorized form automation testing only. Do not use it for:

- Mass account creation on third-party services
- Circumventing payment or trial limitations
- Spam, abuse, credential harvesting or platform evasion

## Reporting issues

If you find sensitive data accidentally committed, remove it immediately and rotate any affected credentials.
